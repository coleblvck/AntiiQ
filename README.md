## Features

- Offline
- It plays audio
- Directory selection
- Customizable behavior
- Backup and restore user preferences and Playlists
- Equalizer
- Pitch, gain and speed tweaks
- Embedded covers support
- Search functionality
- Favourites
- Global Selection
